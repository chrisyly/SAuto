\doxysubsection{Detailed Description}
Modules as well as C\+LI tools for remote M\+XA box control. 

This modules contains the library for remote controling the M\+XA box and it is also a stand alone command line tool The library has to work with a proper configuration file in order to have the ip and port for the remote devices
